Create  Custom  Bootable  Program:


  • Write code in assembly language (ex: boot.asm)
  • Convert .asm to .bin (ex: boot.asm)
  • Create .iso file from .bin and now add .iso file in the newly created VM.

1. Write Code in Assembly Language (boot.asm)


              Goal: Write the assembly code in code Editor and save it as filename.asm for the custom bootloader. 

; boot.asm
[bits 16] ; Set to 16-bit mode (real mode)
[org 0x7c00] ; Set origin to 0x7c00, where BIOS loads the bootloader

start:
mov ah, 0x0e ; BIOS teletype function to print characters
mov al, 'H' ; Print the character 'H'
int 0x10 ; Call interrupt 0x10 (BIOS video interrupt)
mov al, 'i' ; Print the character 'i'
int 0x10

jmp $ ; Infinite loop to halt execution

times 510 - ($ - $$) db 0 ; Pad the bootloader to 512 bytes
dw 0xAA55 ; Boot sector signature
simple bootloader that prints "Hi" on the screen


 2. Convert the .asm File to .bin (boot.asm → boot.bin)


                   Goal: Compile your assembly code into a raw binary file that can be used as a bootloader.

  • Download NASM:  https://www.nasm.us/
  • verify the Installation: 
nasm -v
  • Open CMD  → Navigate to the directory where boot.asm is located → Run the following command: 
nasm -f bin -o boot.bin boot.asm
How to check that .bin file whether it contains that magic nunber(AA55) or not?

Install HxD https://mh-nexus.de/en/hxd/
Run .exe file and open the binary file from HxD and check the magic no.

 3. Create the .iso File from .bin (boot.bin → bootable.iso)


                Goal: Create a bootable ISO file that contains boot.bin file. 

Using ImgBurn:

  • Open ImgBurn and select Create image file from files/folders.
